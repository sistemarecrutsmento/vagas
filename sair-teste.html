Teste 3: Ver qual página está carregando

Se você está vendo o Sair numa página mas não em outras, anote qual funciona e qual não funciona.

  1. index.html → lista de vagas
  2. painel.html → painel / perfil
  3. perfil.html → editar perfil
  4. vaga.html → detalhes de uma vaga
  5. documentos.html → documentos
  6. inscricao.html → inscrição

Abra cada uma, clique no ☰ e veja se o SAIR aparece embaixo.

📱 Se estiver no CELULAR:

  1. Abra o menu do navegador (3 pontinhos)
  2. Configurações → Privacidade → Limpar dados de navegação
  3. Marque "Cache" e "Cookies"
  4. Volte e abra a página de novo
  5. Ou use aba anônima/privativa (Ctrl+Shift+N no PC)
